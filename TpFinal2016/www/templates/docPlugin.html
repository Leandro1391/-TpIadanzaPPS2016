Documentación de Plugins

1) $cordovaCamera : Se usa en la alta y modificación ya sea usuario o producto para así obtener una foto, se ejecuta cuando se hace click en icono de foto.

2) $cordovaFileTransfer : Se usa en la alta y modificación ya sea usuario o producto para poder enviar el archivo de la foto tomada del plugin $cordovaCamera al servidor.

3) $cordovaBarcodeScanner : Se ejecuta en la grilla producto para buscarlo al presionar el botón Escanear y en el alta o modificación del producto con un botón.

4) $cordovaVibration : Se ejecuta cuando se abre el mapa de google y notifica al usuario está a una distancia menor o igual de 100 mts.

5) $cordovaDevice :Se ejecuta en el menú documentación y sierve para informar si el celular cumple con los requirimientos mínimos para que funcione.

6) $cordovaToast : Se ejecuta cuando el usuario se logue correctamente mostrando un cartel diciendo:"Bienvenido", en el formulario de alta de usuario y modificar para informar que solamente se puede ingresar números en el campo código de barras cuando se toca el campo gracias al evento on-touch, en el alta y modificación de usuario indicando al campo contraseña y nombre la cantidad de caracteres que se puede ingresar.

7) $cordovaNativeAudio : idem $cordovaVibration.

8) $cordovaGeolocation : Se ejecuta cuando se abre el mapa de google maps para conocer la ubicación del dispositivo
